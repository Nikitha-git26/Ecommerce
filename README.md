FashionStore - Java Web Based E-Commerce Application using JSP, Servlets, JDBC and MySQL

I developed a Java-based Fashion E-Commerce web application using JSP, Servlets, JDBC, Maven, and Apache Tomcat. The application follows an MVC-style architecture with separate Controller, DAO, Model, and View layers. The project is version-controlled using Git and hosted on GitHub for source code management and collaboration.


# FashionStore

FashionStore is a Java-based E-Commerce web application developed using:

- Java
- JSP
- Servlets
- JDBC
- Maven
- Tomcat Server
- MySQL Database

## Features

- User Registration
- User Login
- Product Catalog
- Shopping Cart
- Checkout
- Order Placement
- Style Quiz Recommendation

## Technologies Used

- Java
- JSP
- Servlet
- JDBC
- MySQL
- Apache Tomcat
- Maven

## Run Project

1. Clone Repository
2. Configure MySQL Database
3. Build using Maven
4. Deploy WAR file to Tomcat
5. Open:
   http://localhost:9090/FashionStore
